# Data-Unity 

This repository is built for the **Data Mining** competition track in **UNITY Competition #14 (UNY National IT Competition) 2026**, hosted by Universitas Negeri Yogyakarta.

## Competition Theme
*"Innovating Beyond Limits: Technology for a Smarter Nation"*

## Team Information
* **Team Name:** ayamBalapBP
* **Institution:** Diponegoro University

## Repository Structure
* **`Modelling/`** : Scripts and notebooks for model training, hyperparameter tuning, and evaluation.
* **`Proposal/`** : Technical report, administrative documents, and guidelines based on `GUIDEBOOK UNITY`.
* **`scrapYT/`** : Web scraping pipeline used for gathering data from YouTube.
* **`data1/`, `data2/`, `data3/`** : Directories storing raw data segments.
* **`dataMerge/`** : Combined, cleaned, and pre-processed datasets ready for analysis.
